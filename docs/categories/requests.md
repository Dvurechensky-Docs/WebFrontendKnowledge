### 🔹 **aja.js**

👉 Простая библиотека для AJAX

- JSON / JSONP
- минимализм

💡 Для:

- простых запросов

⚠️ сейчас почти не используется

---

### 🔹 **axios**

👉 Самый популярный HTTP-клиент

- Promise-based
- interceptors
- удобный API

💡 Использование:

- REST API
- авторизация
- загрузка данных

🔥 стандарт де-факто

---

### 🔹 **d3-request**

👉 часть D3

- удобная альтернатива XMLHttpRequest

💡 Для:

- работы с данными в D3

---

### 🔹 **fetch**

👉 нативный API браузера

- встроен в JS
- Promise

💡 Использование:

- стандартный способ

🔥 сейчас основной вариант

---

### 🔹 **intercooler.js**

👉 AJAX через HTML

- без JS
- атрибуты в HTML

💡 Для:

- простых приложений

⚠️ нишевая штука

---

### 🔹 **qwest**

👉 лёгкая AJAX библиотека

- promises
- XHR2

💡 минимализм

---

### 🔹 **reqwest**

👉 старая AJAX библиотека
💡 сейчас почти не используется ⚠️

---

### 🔹 **superagent**

👉 мощный HTTP клиент

- цепочки
- гибкость

💡 альтернатива axios
