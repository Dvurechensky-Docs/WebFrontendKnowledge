### 🔹 **bean**

👉 Простое API для событий

- add/remove listeners
- удобнее, чем чистый JS

💡 Для:

- работы с DOM событиями

⚠️ сейчас почти не используется

---

### 🔹 **gator.js**

👉 Делегирование событий

- один обработчик → много элементов
- экономия ресурсов

💡 Для:

- списков
- динамического UI

---

### 🔹 **EventEmitter2**

👉 Продвинутый event emitter

- поддержка wildcard (`user.*`)
- сложные события

💡 Для:

- больших систем
- сложной логики 🔥

---

### 🔹 **mitt**

👉 Очень маленький pub/sub

- минимализм
- быстрый

💡 Для:

- простых событий
- коммуникации компонентов ⚡
