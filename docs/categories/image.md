### 🔹 **CamanJS**

👉 Мощная библиотека для обработки изображений

- фильтры (яркость, контраст, виньетка)
- работа через canvas
- цепочки эффектов

💡 Использование:

- редактирование изображений
- фильтры “как в Instagram”
- предпросмотр

🔥 Очень удобная для UI

---

### 🔹 **GrafiJS**

👉 Базовые функции обработки

- blur
- grayscale
- resize

💡 Использование:

- простая обработка
- подготовка изображений

⚡ Лёгкая и простая

---

### 🔹 **Smartcrop**

👉 “умная” обрезка изображения

- определяет важную область
- режет без потери смысла

💡 Использование:

- аватары
- превью
- карточки

🔥 Очень полезно для UX
