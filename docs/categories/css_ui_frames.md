### 🔹 **bootflat**

👉 Bootstrap с “плоским” дизайном (flat UI)

- без теней, без градиентов
- выглядит минималистично и современно
  💡 Подходит для: админок, лендингов

---

### 🔹 **Bootswatch**

👉 Коллекция тем для Bootstrap

- просто меняешь стиль без переписывания кода
- куча готовых дизайнов
  💡 Быстро кастомизировать UI

---

### 🔹 **bulma**

👉 CSS-фреймворк на Flexbox

- без JS, только стили
- удобная сетка и компоненты
  💡 Отлично для чистого фронта без React/Vue

---

### 🔹 **cardinal**

👉 Маленький mobile-first фреймворк

- сначала мобильная версия
- лёгкий и быстрый
  💡 Подходит для PWA и мобильных сайтов

---

### 🔹 **caramel**

👉 Простой и современный CSS-фреймворк

- адаптивный
- чистый дизайн
  💡 Для быстрых проектов без перегруза

---

### 🔹 **corpus**

👉 Набор CSS-утилит и компонентов

- не полноценный фреймворк
- скорее “toolbox”
  💡 Когда нужен контроль и кастом

---

### 🔹 **HTML Starterkit**

👉 HTML5 шаблон для старта

- уже есть структура проекта
- responsive из коробки
  💡 Быстрый старт сайта

---

### 🔹 **kube**

👉 Минималистичный веб-фреймворк

- лёгкий
- удобная типографика
  💡 Для блогов и простых сайтов

---

### 🔹 **Material Design Lite (MDL)**

👉 Компоненты Material Design от Google

- кнопки, карточки, формы
- без тяжёлых зависимостей
  💡 Старый, но понятный

---

### 🔹 **materialize**

👉 Полноценный Material Design фреймворк

- много компонентов
- есть JS
  💡 Для красивых UI “как у Android”

---

### 🔹 **Metro UI**

👉 UI в стиле Windows Metro

- плитки
- flat + строгий стиль
  💡 Под Windows-подобные интерфейсы

---

### 🔹 **milligram**

👉 Минимальный CSS для старта

- почти “голый” стиль
- очень лёгкий
  💡 Быстрое прототипирование ⚡

---

### 🔹 **mini.css**

👉 Минималистичный адаптивный фреймворк

- не навязывает стиль
- лёгкий
  💡 Для кастомных дизайнов

---

### 🔹 **mui**

👉 Лёгкий Material Design фреймворк

- HTML + CSS + немного JS
  💡 Упрощённый аналог Material UI

---

### 🔹 **NES.css**

👉 CSS в стиле Nintendo (ретро 🎮)

- пиксельный дизайн
- выглядит как старые игры
  💡 Для уникальных интерфейсов

---

### 🔹 **paperCSS**

👉 “Бумажный” стиль

- будто нарисовано от руки ✏️
- неформальный дизайн
  💡 Для творческих проектов

---

### 🔹 **papier**

👉 Минималистичный Material-подобный стиль

- лёгкий
- аккуратный UI
  💡 Когда нужен Material без тяжести

---

### 🔹 **pavilion**

👉 Ненавязчивый CSS-фреймворк

- не диктует стиль
- гибкий
  💡 Под кастомную дизайн-систему

---

### 🔹 **picnicss**

👉 Лёгкая UI библиотека

- простые компоненты
- быстро подключается
  💡 Для небольших проектов

---

### 🔹 **pure**

👉 Набор маленьких CSS-модулей от Yahoo

- сетка
- формы
- кнопки
  💡 Берёшь только нужное

---

### 🔹 **semantic-ui**

👉 UI-фреймворк с “человеческими” классами

- readable код (`ui button`)
- куча компонентов
  💡 Удобно, но тяжеловат

---

### 🔹 **Tailwind CSS**

👉 Utility-first фреймворк

- пишешь классы прямо в HTML
- максимальная гибкость
  💡 Сейчас стандарт де-факто 🔥

---

### 🔹 **tachyons**

👉 Модульный utility-фреймворк

- похож на Tailwind
- атомарные классы
  💡 Быстрый и лёгкий

---

### 🔹 **tacit**

👉 CSS без классов 😄

- просто пишешь HTML
- стиль применяется автоматически
  💡 Минимум усилий

---

### 🔹 **uikit**

👉 Лёгкий модульный фреймворк

- компоненты + JS
- гибкий
  💡 Альтернатива Bootstrap

---

### 🔹 **wing**

👉 Минимальный CSS-фреймворк

- почти без зависимостей
- очень простой
  💡 Для MVP и прототипов
