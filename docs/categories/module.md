### 🔹 **Browserify**

👉 Позволяет использовать `require()` в браузере

- как в Node.js
- объединяет файлы

💡 Исторически важен
⚠️ сейчас почти не используется

---

### 🔹 **RequireJS**

👉 AMD загрузчик

- асинхронная загрузка модулей

💡 Старый подход ⚠️
👉 был популярен до ES6

---

### 🔹 **Rollup**

👉 Современный bundler

- ES6 модули
- tree shaking (удаляет лишний код)

💡 Использование:

- библиотеки
- лёгкие проекты

🔥 очень чистый output

---

### 🔹 **StealJS**

👉 загрузчик + сборщик

- зависимости
- билд

💡 редко используется сейчас

---

### 🔹 **SystemJS**

👉 универсальный загрузчик

- ES6 / AMD / CommonJS

💡 для сложных систем
⚠️ не основной сегодня

---

### 🔹 **Webpack**

👉 главный bundler индустрии

- сборка проекта
- плагины
- dev server

💡 Использование:

- React
- Vue
- большие проекты

🔥 стандарт де-факто
