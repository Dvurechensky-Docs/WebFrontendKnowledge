### 🔹 **D3**

👉 мощнейшая библиотека визуализации

- SVG
- data-driven подход

💡 Для:

- графиков
- аналитики

### 🔹 **EaselJS**

👉 Canvas 2D интерактив

- анимации
- события

💡 Для:

- игр
- UI

### 🔹 **Fabric.js**

👉 работа с canvas как с объектами

- редактирование
- drag & drop
- SVG ↔ Canvas

💡 Для:

- редакторов
- графических инструментов 🔥

- простых визуализаций
