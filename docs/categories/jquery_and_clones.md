### 🔹 **bliss**

👉 Набор утилит поверх обычного JS

- добавляет “сахар” (упрощает синтаксис)
- не заменяет полностью jQuery
  💡 Для тех, кто хочет писать на чистом JS, но проще

---

### 🔹 **cash**

👉 Лёгкий аналог jQuery

- почти такой же синтаксис (`$('.class')`)
- но использует современные API браузера
  💡 Отличный вариант вместо jQuery ⚡

---

### 🔹 **jQuery**

👉 Классическая библиотека

- работа с DOM
- AJAX
- события
  💡 Раньше стандарт, сейчас устаревает

---

### 🔹 **nanoJS**

👉 Минимальный JS для работы с DOM

- очень маленький
- только базовые операции
  💡 Когда нужен ультра-лёгкий код

---

### 🔹 **NodeList.js**

👉 Упрощает работу с нативным DOM

- делает API похожим на jQuery
  💡 Если хочешь остаться на чистом JS

---

### 🔹 **Selector**

👉 Лёгкая замена jQuery

- минимальный функционал
- быстрый
  💡 Для небольших проектов

---

### 🔹 **umbrella**

👉 Библиотека для DOM + события + AJAX

- компактная
- почти как jQuery, но легче
  💡 Хороший баланс возможностей

---

### 🔹 **zeptojs**

👉 Почти полный аналог jQuery

- совместимый API
- сильно меньше размер
  💡 Часто используется в мобильных проектах
